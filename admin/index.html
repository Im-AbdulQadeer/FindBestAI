🐙 Step 1 — Upload to GitHub Repository

Upload all your site files to a GitHub repository. Cloudflare Pages will automatically deploy every time you push changes.

1. Create a GitHub account at github.com 2. Create a new repository: "aihub" (public or private) 3. Upload ALL files keeping the exact folder structure: ├── index.html ├── assets/ │ ├── css/main.css │ └── js/ (data.js, app.js, nav.js) ├── tools/index.html ├── compare/index.html ├── blog/index.html │ ├── best-ai-video-generator-2026/index.html │ └── [other articles]/index.html ├── about/index.html ├── contact/index.html ├── privacy/index.html ├── terms/index.html ├── cookies/index.html ├── admin/index.html └── _redirects

To add a new article later: create blog/[slug]/index.html and push to GitHub.

☁️ Step 2 — Connect Cloudflare Pages

Cloudflare Pages is the fastest free static hosting with global CDN. Connect it to your GitHub repo in 3 clicks.

1. Go to dash.cloudflare.com → Pages → Create a project 2. Connect your GitHub account 3. Select your "aihub" repository 4. Build settings: - Framework preset: None - Build command: (leave empty) - Build output directory: / (root) 5. Click "Save and Deploy" 6. Your site is live at: [project-name].pages.dev

Every git push to your main branch auto-deploys in ~30 seconds. Use the _redirects file for clean URLs.

🌐 Step 3 — Connect Your Custom Domain

1. In Cloudflare Pages → your project → Custom domains 2. Click "Set up a custom domain" 3. Enter your domain (e.g. aihub.tools) 4. If your domain is on Cloudflare DNS: - DNS records are added automatically ✓ 5. If domain is on another registrar: - Add a CNAME: @ → [project].pages.dev - Or point nameservers to Cloudflare 6. SSL certificate issued automatically (free)

📄 _redirects File (Required for Clean URLs)

This file is already in your project root. It tells Cloudflare Pages how to handle URL routing.

🔍 Google Search Console — Submit & Index Pages

Google Search Console lets you submit your sitemap and request indexing for individual URLs. This is the fastest way to get pages into Google.

1. Go to search.google.com/search-console 2. Add property: select "URL prefix" → enter https://yourdomain.com 3. Verify ownership: HTML tag method — copy the tag into your <head> (Already added to index.html as google-site-verification meta tag) 4. Submit sitemap: Sitemaps → Enter sitemap.xml → Submit 5. Request indexing for new pages: URL Inspection → paste URL → "Request Indexing" Do this for every new article after publishing: - https://yourdomain.com/blog/[slug]/

🌐 Bing Webmaster Tools

1. Go to bing.com/webmasters 2. Add your site and verify ownership 3. Submit sitemap.xml 4. Use URL Inspection to request indexing of new pages (Bing often indexes faster than Google for new sites)
